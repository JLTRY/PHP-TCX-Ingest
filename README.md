PHP-TCX-Ingest
==============

PHP TPX-Ingest is a simple PHP class designed to ingest a basic (TCX) file and extract data/stats from the tracks within. 

It has been inspired/derived from PHP-GPX-INJECT https://github.com/bentasker/PHP-GPX-Ingest.git


Features
----------

The class has the following features

 - Import basic TCX files
 - Generate stats for each track, and maintain global stats
